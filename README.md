#Shopping Cart Items (React)
This React code displays shopping cart items with detailed information and allows users to remove products from the cart.

##Features
Show product details (code, name, size, color)
Calculate total price based on quantity
Remove items dynamically
Clean UI using Tailwind CSS
Dynamic rendering with map metod

##Built With
React
JavaScript
Tailwind CSS

##Purpose
Practice rendering lists, handling user actions, and calculating values inside React.
